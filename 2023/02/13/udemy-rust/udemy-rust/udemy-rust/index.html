what is rust

  • 现代编程语言,可以解决过去编程语言的问题;
  • 无null(有丰富的类型系统,无引用与指针);
  • 无异常(强制的异常处理);
  • 良好的包管理和构建系统;
  • 无数据竞争(利于编写异步代码)

    installing rust

    cargo

    创建项目

    # 如果是--lib,则声明为一个类库,不可执行
cargo new example
cd example 
git init

    添加依赖

    Cargo.toml

    [package]
name = "example"
version = "0.1.0"
edition = "2021" 
[dependencies]
rand = "0.7"

    构建

    cargo build

    运行项目

    cargo run

    安装依赖

    cargo install cargo-expand

    如果报这个错,则安装工具链

    配置cargo镜像(网上搜的)

    [source.crates-io]
registry = "https://github.com/rust-lang/crates.io-index"
#指定镜像
replace-with = 'tuna'
#清华大学
[source.tuna]
registry = "https://mirrors.tuna.tsinghua.edu.cn/git/crates.io-index.git"

数据科学

从数据提取价值

  1. data engineer 收集数据
  2. data scientist 分析,可视化
  3. machine learning expert 智能算法分析新数据, 得出结论

用数据科学解决问题 + 线性回归

线性回归 -> 建模, 发现潜在联系

数据分析科学家会:

  1. formulate question 提出要回答的问题
  2. gather data 收集(有助于回答问题的)数据
  3. clean data 清理无用数据(噪声)
  4. explore & visualise 探究与可视化
  5. train algorithm 训练算法 (本节的算法是线性回归)
  6. evaluate 评估结果

step1 一个好问题

  • 我的电影能赚多少💴 => 不是好问题, 太模糊fake了, 金钱money的定义?
  • 我的电影可以赚多少收入(revenue) => 比金钱money更具体, 可以更好地定量分析(测量, 追踪)
  • 我们可以使用电影预算来预测电影收入吗 => 多了一个x自变量, 他是可测试的(可以检查预算x和收入y是否存在关系) , 这里的y就是dependent variable 或 机器学习中的target, 而x则是 independent variable 或 机器学习中的 feature

step2 收集数据

  1. data on feature 自变量x 电影预算
  2. data on target 因变量y 电影收入

去哪获取数据? the numbers

数据 -> 排序和筛选 -> 不需要包含标题 -> 排序列 -> 排序

为0的数据可能是因为从未上映(没有数据) 或 数据有问题/争议

step3 清理数据

  1. 选取所有含0数据的项 -> 右键删除

  2. 删除前三列和最后一列,留下Production Budget和Worldwide Gross, 因为对计算没有用

  3. 将$x,000,000 格式转为纯数字格式

  4. 重命名列名(去掉特殊符号)
